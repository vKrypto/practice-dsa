Given a set of intervals, find out if any two intervals overlap.


#### Example 1
Intervals: [[1,4], [2,5], [7,9]]
Output: true
Explanation: Intervals [1,4] and [2,5] overlap

Explanation: Intervals [6,7] and [9,12] do not overlap
